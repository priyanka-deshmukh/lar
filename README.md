# @predash/lar

[![npm (scoped)](https://img.shields.io/npm/v/@predash/lar.svg)](https://www.npmjs.com/package/@predash/lar)


Performs d number of left rotation operation on array and returns output.

## Install

```
$ npm i @predash/lar
```

## Usage

```js
const lar = require("@predash/lar");

tiny(3,[1,2,3,4]);
//=> [4,1,2,3]

```
